Back to the Wild Game Manual

1. Overview

Back to the Wild is a 3D platformer game set in a zoo in which you start off taking control of a raccoon who decides to rescue the animals within the zoo. The player must solve puzzles and maneuver around different animal's exhibits in order to help the animals escape. The player can link up with animals in a specific level in order to use their special abilities.

2. Gameplay

Game Controls

  • WASD: Move forward, left, down, and right respectively
  • Spacebar: Jump
  • E: The animal's special ability
  • L: Link with another animal
  • 0, 1, 2, 3: Switch between different animals after you have linked up with them
  • I: Take an in game screenshot
  • P: Start recording game play, press again to stop

Game Mechanics

  • The player can switch between different animals that have different abilities
  • The default character is the raccoon, which can interact with puzzles requiring hands
  • The objective is to get keys and help other animals escape
  • This would be a 3rd person, over the shoulder game unless you are solving a 2D puzzle
  • Trash cans are save points throughout the level and the player can also obtain points by finding candy bars/food from trash bins

Puzzles

  • A wire puzzle that would trigger certain objects to move or open like a draw bridge
  • A lever puzzle to raise and lower platforms to the correct height
  • A box puzzle where the player would have to push around boxes in order to reach certain areas
  • A laser puzzle where you have the align the mirrors correctly
  • A keypad puzzle where you have to find the code
  • A pressure plate puzzle where certain animals such as an elephant would need to stand in place for doors to open

3. Characters

  • Raccoon: Main character of the game. Is able to jump, move around, and solve puzzles that require hands. Will be available to use in all levels.
  • Safari Animals
    • Giraffe: Can reach and interact with tall objects.
    • Zebra: Can dash allowing it to reach platforms that are out of reach for the raccoon.
    • Hippo: Can move objects as well as activate pressure plates.
    • Lion: Can attack enemies.
  • Desert Animals
    • Scorpion: Can place a single block that can be used to jump to higher places.
    • Desert Hare: Can jump higher than other animals.
    • Elephant: Can move objects as well as activate pressure plates.
    • Fire Salamander: Can spit fire.
  • Polar Animals
    • Penguin: Can throw snowballs.
    • Polar Bear: Can move objects as well as activate pressure plates.
    • Reindeer: Can fly for a short duration.
    • Wolf: Can attack enemies.
